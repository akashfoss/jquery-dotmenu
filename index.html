Dependencies

jQuery
1.6.2 or higher, not tested on lower
www.jquery.com
jQuery-UI (highly recommended)
1.8 or higher, not tested on lower
www.jqueryui.com

Install

  1. Get the latest jQuery (and optionally, jQuery-UI); this was developed using 1.6.2, but there may possibly be ways to make it work in an earlier version. If you feel like making that happen, fork it and send me a request!
  2. Get the JS file and the default CSS file that goes with it. Put them in a location that's accessible by your intended webpage.
  3. Reference the files in your page.
  4. Follow the usage instructions below!
